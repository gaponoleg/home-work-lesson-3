## Списки

Чтобы добавить ненумированные списки, необходимо пункты выделить звездочкой (*).
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
